# ETitle

Defines the document's title that browsers show in the title bar or a page's tab.

### Import

```js
import { ETitle } from 'react-mailkit';
```

### Examples

```html
<EMail>
  <EHead>
    <ETitle>Hello MJML</ETitle>
  </EHead>
  <EBody>
    <ESection>
      <EColumn>
        <EText> Hello World! </EText>
      </EColumn>
    </ESection>
  </EBody>
</EMail>
```
