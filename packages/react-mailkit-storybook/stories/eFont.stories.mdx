# EFont

This component imports fonts. The component has effect only if the template uses the font, too. The href attribute points to a hosted css file; that file contains a @font-face declaration.
Example: https://fonts .googleapis.com/css?family=Raleway

### Import

```js
import { EFont } from 'react-mailkit';
```

### Examples

```html
<EMail>
  <EHead>
    <EFont
      name="Raleway"
      href="https://fonts.googleapis.com/css?family=Raleway"
    />
  </EHead>
  <EBody>
    <ESection>
      <EColumn>
        <EText font-family="Raleway, Arial"> Hello World! </EText>
      </EColumn>
    </ESection>
  </EBody>
</EMail>
```
