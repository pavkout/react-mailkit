# EPreview

This component allows you to set the preview that will be displayed in the inbox of the recipient.

EPreview doesn't support any attribute.

### Import

```js
import { EPreview } from 'react-mailkit';
```

### Examples

```html
<EMail>
  <EHead>
    <EPreview>Hello MJML</EPreview>
  </EHead>
  <EBody>
    <ESection>
      <EColumn>
        <EText> Hello World! </EText>
      </EColumn>
    </ESection>
  </EBody>
</EMail>
```
