# EBreakpoint

This component allows you to control on which breakpoint the layout should go desktop/mobile.

### Import

```js
import { EBreakpoint } from 'react-mailkit';
```

### Examples

```html
<EMail>
  <EHead>
    <EBreakpoint width="320px" />
  </EHead>
  <EBody>
    <ESection>
      <EColumn>
        <EText> Hello World! </EText>
      </EColumn>
    </ESection>
  </EBody>
</EMail>
```
